공지사항

한글패치 외 아래 확장 프로그램 추가설치를 권장함. 단 게임 중 로딩중에 멈춤현상이 발생하는 경우 아래 확장 프로그램이 문제일 수 있음. 아래 확장 프로그램이 문제가 있는 경우 한글패치만 단독으로 사용하길 권장함

-> 뉴비인데 뭔말인지 잘 모르겠으면 https://github.com/Gnbrkm41/CYKViewer/releases 그냥 이거 다운받아서 쓰고 아래 가이드 내용 다 무시

 

22.04.29 크롬 확장 프로그램 신규 추가 알림

https://gall.dcinside.com/mgallery/board/view/?id=shinycolors&no=1060179

아래부터 진짜 한글패치 가이드

PC

  1. Chrome 브라우저에서 Violentmonkey 확장 프로그램을 설치합니다.

    https://chrome.google.com/webstore/detail/violentmonkey/jinjaccalgkegednnccohejagnlnfdag

  2. 확장 프로그램이 설치된 후 https://greasyfork.org/ko/scripts/404372-%EC%83%A4%EB%8B%88%EB%A7%88%EC%8A%A4-%ED%95%9C%EA%B8%80-%ED%8C%A8%EC%B9%98-%EC%9E%84%EC%8B%9C의 주소에 접속하여 설치를 진행합니다.

  3. 게임 사이트 https://shinycolors.enza.fun에 접속하여 한글 패치가 정상적으로 진행되었는지 확인합니다.

  4. 해당 한글패치는 자주 버전 업데이트를 합니다. 게임 갱신일(가챠 갱신일)때 항상 업데이트 여부를 확인한 후 업데이트를 수행하시기 바랍니다.

  5. 브라우저 우측 상단의 Violentmonkey에서 한글패치의 기타 추가기능(문서 맨 아래 참고)을 이용할 수 있습니다.

  6. 가끔 이용중 버그가 있으면 브라우저에서 f5(새로고침)를 눌러주시고, 버그가 여전히 존재하면 ctrl + f5(강력 새로고침)를 눌르면 대부분의 문제는 해결됩니다.

     

-> PC 전용 게임 실행기를 사용하면 업데이트가 자동으로 적용되어 편하게 한글패치를 적용하여 게임 진행이 가능합니다.

안드로이드

  1. 기본적으로 Kiwi 브라우저를 설치하는 거 외에는 PC와 동일합니다.

  2. 구글스토어에서 Kiwi 브라우저를 설치합니다.

    https://play.google.com/store/apps/details?id=com.kiwibrowser.browser

    팁 : 최근 kiwi,크롬,삼성브라우저 등 안드로이드 브라우저에서 심한 랙이 발생하는 현상이 있습니다. 구글 스토어 대신 아래 링크에서 APK 파일을 다운받고 Kiwi 브라우저(구버전) 설치를 추천합니다.

    https://github.com/kiwibrowser/src/releases/download/1130924267/com.kiwibrowser.browser.classic-1130924267-arm64-signed.apk

    -> 구버전 kiwi가 최근에 안되는 현상이 있습니다. 만약 구버전 kiwi를 쓰려는 경우 tampermonkey를 쓰시기 바랍니다.

  3. Kiwi 브라우저를 실행한 후 Kiwi 브라우저 내에서 Violentmonkey 확장 프로그램을 설치합니다.

    https://chrome.google.com/webstore/detail/violentmonkey/jinjaccalgkegednnccohejagnlnfdag

  4. 확장 프로그램이 설치된 후 https://greasyfork.org/ko/scripts/404372-%EC%83%A4%EB%8B%88%EB%A7%88%EC%8A%A4-%ED%95%9C%EA%B8%80-%ED%8C%A8%EC%B9%98-%EC%9E%84%EC%8B%9C의 주소에 접속하여 설치를 진행합니다.

  5. 게임 사이트 https://shinycolors.enza.fun에 접속하여 한글 패치가 정상적으로 진행되었는지 확인합니다.

  6. 해당 한글패치는 자주 버전 업데이트를 합니다. 게임 갱신일(가챠 갱신일)때 항상 업데이트 여부를 확인한 후 업데이트를 수행하시기 바랍니다.

  7. Kiwi 브라우저 설정에서 '홈화면에 추가' 누르고 홈 화면에서 추가된 아이콘을 이용하면 풀스크린으로 게임을 플레이할 수 있습니다.

  8. Kiwi 브라우저 설정에서 설치된 Violentmonkey로 이동하여 한글패치의 기타 추가기능(문서 맨 아래 참고)을 이용할 수 있습니다. 대부분 PC에서 사용하는 기능이므로 모바일에서 사용은 삼가해주시기 바랍니다. 다만 서버 바꾸기 기능은 사용해볼 만 합니다.

-> alook 브라우저 사용시 오른쪽 위 메뉴(三)에서 비디오 플로팅을 꼭 끄시기 바랍니다

한글패치 기타 추가기능(PC크롬에서 사용, 안드로이드 Kiwi에서도 사용가능함)

  • Violentmonkey에서 사용 가능
  1. 개발자 모드 : 프로그래머를 위한 기능이니 일반사용자는 관심두지말고 키지도 말 것
  2. 백그라운드BGM : 백그라운드 상태에서도 게임 BGM이 재생됩니다.
  3. 샤니위키 : 샤니위키 사이트에 접속합니다.
  4. 업데이트 : 스크립트 업데이트를 위해 스크립트 URL에 접속합니다.
  5. 커뮤추출 : 한글패치를 위해 커뮤를 추출합니다.
  6. 서버바꾸기 : 한글패치 서버를 바꿉니다. shinymaskr.work로 바꿀 시 높을 확률로 게임로딩속도가 빨라지고 데이터 소모량이 감소합니다.

-> 1,2,5는 한번 누르면 기능 ON, 다시 한번 누르면 기능 OFF

-> 6은 누르면 접속 서버가 바뀌며, 게임 새로 접속이 필요합니다.
